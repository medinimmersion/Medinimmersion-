# RIHLA by MédinImmersion — Game Design Bible

Reference documentation for RIHLA, an immersive educational game concept in which the player travels
through a series of countries toward Médine while learning Arabic. These documents describe the vision,
art direction, characters, academy setting, and onboarding flow underpinning the concept; they are
provided as design reference and are not yet reflected in the current web application codebase.
