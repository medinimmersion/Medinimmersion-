# RIHLA by MédinImmersion — Game Design Bible

## Document 027 – Interface utilisateur

Version 1.0

### Philosophie

L'interface doit sembler intégrée au monde et non superposée.

### Éléments

Carnet, passeport, objectifs du jour, boutons sobres, textes lisibles.

### Règles

Limiter les fenêtres surgissantes et privilégier les interactions dans le décor.
