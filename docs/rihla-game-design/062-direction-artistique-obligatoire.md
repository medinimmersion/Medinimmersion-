# 062 — Direction artistique obligatoire de RYHLA

## Niveau de qualité minimum

Le résultat graphique de RYHLA devra donner l'impression d'un véritable jeu commercial.

Aucun élément du jeu ne devra ressembler à un prototype, une démonstration technique ou un projet étudiant.

Le joueur devra avoir l'impression de jouer à un jeu terminé dès les premières minutes.

---

## Interdictions absolues

Les éléments suivants sont strictement interdits :

- ❌ Personnages créés avec des cylindres.
- ❌ Personnages créés avec des cubes.
- ❌ Personnages créés avec des sphères.
- ❌ Corps simplifiés.
- ❌ Personnages sans vêtements modélisés.
- ❌ Personnages low-poly visibles.
- ❌ Placeholders.
- ❌ Assets de démonstration Unity.
- ❌ Assets "Prototype".
- ❌ Sol totalement plat.
- ❌ Terrain vide.
- ❌ Couleurs unies.
- ❌ Matériaux par défaut Unity.
- ❌ Texture unique sur tout le sol.
- ❌ Bâtiments sans détails.
- ❌ Murs sans relief.
- ❌ Fenêtres dessinées.
- ❌ Portes dessinées.
- ❌ Arbres low-poly.
- ❌ Éclairage Unity par défaut.
- ❌ Interface de démonstration.
- ❌ Icônes temporaires.
- ❌ Police générique.

---

## Qualité attendue

Chaque scène devra être digne d'une capture d'écran officielle.

Chaque image devra pouvoir être publiée sur le Play Store.

Chaque environnement devra donner envie d'être exploré.

Le joueur devra immédiatement ressentir une impression de qualité professionnelle.

---

## Personnages

Les personnages devront être créés à partir de véritables modèles humains.

Le corps devra posséder des proportions réalistes.

Le qamis devra être un véritable vêtement modélisé.

Le ghutra devra être un véritable tissu.

L'agal devra être modélisé séparément.

La barbe devra être en véritable géométrie.

Les mains devront être entièrement modélisées.

Chaque doigt devra être indépendant.

Les sandales devront être modélisées.

Les vêtements devront posséder :

- coutures
- plis
- épaisseur
- poids
- mouvement naturel

Les tissus devront réagir pendant la marche.

Le personnage devra produire une ombre réaliste.

Le visage restera volontairement lisse conformément à l'identité graphique de RYHLA.

---

## Architecture

Chaque bâtiment devra être inspiré de photographies réelles.

Les proportions devront être respectées.

Les murs devront posséder :

- relief
- profondeur
- matériaux réalistes
- imperfections naturelles

Les portes devront être entièrement modélisées.

Les fenêtres devront posséder une véritable profondeur.

Les colonnes devront être rondes.

Les escaliers devront être utilisables.

Les toitures devront être détaillées.

Les enseignes devront être réalistes.

Les boutiques devront sembler réellement utilisées.

---

## Sol

Le terrain ne devra jamais être vide.

Le sol devra contenir :

- pavés
- sable
- pierre
- terre
- herbe
- végétation

selon le lieu.

Les matériaux devront être différents suivant les quartiers.

---

## Décors

Chaque rue devra contenir de nombreux éléments.

Exemples :

- bancs
- lanternes
- plantes
- arbres
- enseignes
- étals
- caisses
- tapis
- livres
- pots
- fontaines
- panneaux
- décorations
- murs détaillés
- balcons
- arches
- portes
- fenêtres
- volets
- éclairages

Aucune rue ne devra sembler vide.

---

## Mosquées

Toutes les mosquées devront être reproduites à partir de références réelles.

Claude Code devra utiliser plusieurs photographies.

Il devra analyser :

- la façade
- les dimensions
- les matériaux
- les couleurs
- les colonnes
- les minarets
- les dômes
- les cours
- les portes
- les fenêtres
- les lustres
- les tapis
- le mihrab
- le minbar

L'objectif est que le joueur reconnaisse immédiatement la mosquée.

---

## Monuments

Tous les monuments devront être fidèlement reproduits.

Ils devront être construits à partir de photographies, de plans et de vues aériennes lorsqu'elles existent.

---

## Éclairage

Le jeu devra utiliser un éclairage cinématographique.

Aucune scène ne devra utiliser l'éclairage Unity par défaut.

Le soleil devra produire :

- des ombres réalistes
- une lumière chaude
- une lumière indirecte
- une ambiance agréable

Le lever et le coucher du soleil devront être particulièrement soignés.

---

## Caméra

La caméra devra être fluide.

Elle ne devra jamais être rigide.

Les mouvements devront être doux.

Les transitions devront être invisibles.

---

## Végétation

Tous les arbres devront être détaillés.

Les feuilles devront être nombreuses.

Les plantes devront être variées.

Le vent devra provoquer un léger mouvement.

---

## Ambiance

Chaque ville devra sembler vivante.

Le joueur devra ressentir qu'il visite une véritable ville.

Aucun quartier ne devra sembler abandonné sauf si cela est volontaire.

---

## Objectif graphique

Le niveau graphique visé devra être comparable aux meilleurs jeux mobiles modernes.

Le joueur ne devra jamais penser qu'il joue à un prototype.

Chaque nouvel élément ajouté dans RYHLA devra respecter exactement cette direction artistique.

La qualité finale devra être constante dans l'ensemble du jeu, sans exception.
