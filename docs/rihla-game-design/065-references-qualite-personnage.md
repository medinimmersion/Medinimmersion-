# 065 — Références officielles de qualité du personnage principal

Le personnage principal devra atteindre un niveau de qualité comparable aux personnages des jeux suivants.

Ces jeux constituent les références officielles de RYHLA.

## Assassin's Creed Mirage

Référence pour :

- Le réalisme des vêtements.
- Les tissus.
- Les plis.
- Les sandales.
- Les matériaux.
- Les animations.
- Les mouvements.
- Les proportions du corps.
- L'architecture du monde.

Le niveau de finition devra être comparable.

---

## Journey

Référence pour :

- La silhouette.
- L'ambiance.
- Les éclairages.
- La caméra.
- Les déplacements.
- Le sentiment de voyage.

Le personnage devra transmettre la même sérénité.

---

## Kena: Bridge of Spirits

Référence pour :

- Les matériaux.
- Les textures.
- La qualité générale.
- Les animations.
- Les effets de lumière.

Le rendu devra être très propre.

---

## Genshin Impact

Référence uniquement pour :

- La fluidité.
- L'optimisation mobile.
- La qualité des animations.
- La qualité de l'interface.
- Les performances.

Ne jamais reprendre le style anime.

---

## Red Dead Redemption 2

Référence pour :

- Le niveau de détail des vêtements.
- Les mouvements naturels.
- Les animations.
- Les matériaux.
- Les effets de tissu.

Même si RYHLA est un jeu mobile, viser ce niveau de réalisme.

---

## Hogwarts Legacy

Référence pour :

- Les bâtiments.
- Les intérieurs.
- Les bibliothèques.
- Les objets.
- Les salles.
- Les détails des environnements.

---

## Ghost of Tsushima

Référence pour :

- La direction artistique.
- Les couleurs.
- Les paysages.
- Les jeux de lumière.
- Les effets de vent sur les vêtements.
- Les mouvements des tissus.

---

# Objectif final

Le résultat ne devra jamais donner l'impression d'un projet amateur.

Il devra donner l'impression d'un véritable jeu commercial développé par un studio professionnel.

Chaque personnage devra pouvoir apparaître dans une bande-annonce.
