# RIHLA by MédinImmersion — Game Design Bible

## Document 022 – Sauvegarde & Profils

Version 1.0

### Objectif

Chaque élève possède un profil unique synchronisé avec sa progression.

### Contenu

Profil, passeport, statistiques, devoirs, certificats, paramètres.

### Règles

Aucune perte de progression. Synchronisation après chaque étape importante.
