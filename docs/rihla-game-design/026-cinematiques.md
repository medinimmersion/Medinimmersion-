# RIHLA by MédinImmersion — Game Design Bible

## Document 026 – Cinématiques

Version 1.0

### Objectif

Chaque changement de pays ou d'étape importante est introduit par une courte cinématique.

### Principes

Caméra fluide, transitions lentes, aucun effet agressif.

### Moments clés

Arrivée dans un pays, rencontre d'un enseignant, remise de certificat.
