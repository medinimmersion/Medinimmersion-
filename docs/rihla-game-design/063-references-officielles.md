# 063 — Références officielles pour le développement de RYHLA

Afin d'obtenir un résultat de très haute qualité, le développement de RYHLA devra s'inspirer des meilleurs jeux existants, tout en conservant une identité unique et conforme aux objectifs pédagogiques du projet.

Chaque jeu ci-dessous constitue une référence pour un aspect précis du développement.

---

## 1. Journey

Référence principale pour :

- L'ambiance générale.
- La sérénité.
- Les déplacements.
- L'immersion.
- Les paysages.
- Les jeux de lumière.
- Les couleurs.
- Les effets visuels.
- La fluidité des animations.
- La sensation de voyage.

RYHLA devra procurer le même sentiment d'émerveillement et de découverte.

---

## 2. Assassin's Creed Mirage

Référence principale pour :

- L'architecture.
- Les rues.
- Les bâtiments.
- Les marchés.
- Les matériaux.
- Les mosquées.
- Les monuments.
- Les détails des villes.
- Le réalisme des décors.

Toutes les villes de RYHLA devront être inspirées de véritables photographies, comme dans Assassin's Creed Mirage.

Les monuments devront être fidèlement reproduits.

---

## 3. Genshin Impact

Référence principale pour :

- La carte du monde.
- Le système de progression.
- Les zones débloquées progressivement.
- Les déplacements entre les villes.
- L'interface utilisateur.
- Les animations des menus.
- La qualité générale des environnements.

Le joueur devra toujours ressentir l'envie de découvrir la prochaine ville.

---

## 4. Pokémon

Référence principale pour :

- Le système de progression.
- Le déblocage progressif des régions.
- Les quêtes principales.
- Les quêtes secondaires.
- Les récompenses.
- Les objets.
- Le passeport.
- Les voyages entre les pays.

Le joueur ne pourra accéder aux nouvelles régions qu'après avoir validé les étapes pédagogiques précédentes.

---

## 5. Duolingo

Référence principale pour :

- La progression pédagogique.
- Les récompenses.
- Les objectifs quotidiens.
- Les séries.
- Les statistiques.
- Les encouragements.
- Les exercices.

L'apprentissage devra rester motivant sans devenir répétitif.

---

## 6. DragonBox

Référence principale pour :

- L'apprentissage par le jeu.
- Les mécanismes pédagogiques.
- La découverte progressive des notions.

Le joueur devra apprendre sans avoir l'impression d'étudier.

---

## 7. Professor Layton

Référence principale pour :

- Les défis.
- Les énigmes.
- Les interactions avec les personnages.
- Les scènes de dialogue.

Les activités devront stimuler la réflexion.

---

## Direction artistique

Le style graphique devra être un mélange de :

- Journey.
- Assassin's Creed Mirage.
- Genshin Impact.

L'objectif est d'obtenir un rendu :

- Semi-réaliste.
- Cinématographique.
- Chaleureux.
- Très détaillé.
- Immersif.
- Moderne.
- Fluide.
- Élégant.

Aucun élément ne devra être cartoon, low-poly ou ressembler à un prototype.

---

## Architecture

Toutes les villes devront être reproduites à partir de photographies réelles.

Les bâtiments devront respecter l'architecture locale.

Les mosquées devront être fidèlement reproduites.

Les monuments devront être immédiatement reconnaissables.

Chaque pays devra posséder sa propre identité visuelle.

---

## Objectif final

RYHLA devra être un mélange de :

- Journey pour l'ambiance.
- Assassin's Creed Mirage pour les villes.
- Genshin Impact pour l'exploration.
- Pokémon pour la progression.
- Duolingo pour la pédagogie.
- DragonBox pour l'apprentissage ludique.
- Professor Layton pour les défis.

Le résultat final devra être un jeu unique, où le joueur apprend la langue arabe en voyageant dans le monde musulman, accompagné de son oustaz, dans un univers immersif, réaliste et de très haute qualité graphique.
