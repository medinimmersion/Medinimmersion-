# RIHLA by MédinImmersion — Game Design Bible

## Document 015 – Carte du Caire (Fondations)

Version 1.0

### Vision générale

Le Caire est conçu comme une ville pédagogique. Chaque quartier correspond à une étape d'apprentissage. Le joueur n'est jamais perdu : l'oustaz guide toujours le déplacement.

### Carte simplifiée

1. Porte d'arrivée
2. Place centrale
3. Rue des Lettres
4. Chez Alif
5. Chez Ba
6. Bibliothèque
7. Petit marché
8. Jardin de révision
9. Porte de sortie vers la ville suivante

### Règles de progression

- La Rue des Lettres est inaccessible tant que l'introduction n'est pas terminée.
- Chaque nouvelle boutique se débloque après validation de la précédente.
- Les raccourcis apparaissent uniquement après avoir découvert une zone.
- Les révisions peuvent ramener l'élève dans les anciennes boutiques.

### Déplacements

Abdallah marche toujours légèrement devant l'élève. Lorsqu'il souhaite attirer l'attention sur un lieu, il ralentit, se tourne vers l'élève puis reprend sa marche après l'interaction.

### Objectif

La carte doit donner envie d'explorer tout en gardant un parcours pédagogique clair et progressif.
